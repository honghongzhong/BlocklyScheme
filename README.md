# BlocklyScheme

BlocklyScheme is a web-based application that combines Blockly visual programming with Lisp code generation. It allows users to create programs using drag-and-drop blocks and generates corresponding Lisp code.

## Features
- Visual programming with Blockly blocks
- Lisp code generation
- Multilingual support (i18n)
- Custom block definitions

## Project Structure
- `app.js`: Main application logic
- `blocks.js`: Custom Blockly blocks
- `generator.js`: Lisp code generator
- `i18n.js`: Internationalization
- `index.html`: Main web page
- `BiwaScheme.html`: Scheme interpreter integration
- `styles.css`: Stylesheet
- `package.json`: Project metadata and dependencies

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the application:
   ```bash
   npm run start
   ```
3. Open `index.html` in your browser to use BlocklyLisp.
